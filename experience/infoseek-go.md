# Infoseek / Go.com / Disney Internet Group — Staff Engineer, E-Commerce Group

## Context

Designed and developed server-side e-commerce applications and infrastructure for a major media portal (Infoseek became Go.com / Disney Internet Group during this tenure).

## Contributions

- **BxsonMgr** – Designed and developed a Java package for accessing topic ontology information from a remote server, encoded in XML and transported over HTTP.
- **QueryBean** – Designed and developed a Java package providing convenient JDBC result access from within the proprietary Tea templating language, utilizing the JavaBeans design pattern.
- **JTP-Servlet Bridge** – Designed and developed a Servlet engine bridge implementing the Java Servlet 2.1 standard within the proprietary JTP application server.
- **MessagePipeline** – Designed and developed a simple transactional message queue to support application integration. Advocated a message-based architecture throughout the organization.
- **CreditCardAuth** – Designed and developed a credit card authorization package interpreting authorization codes to logical business cases (Fraud / Not Fraud). Used message queues for asynchronous authorizations and relaying results across systems.
- **"Lego" Design Pattern** – Proposed, developed, and evangelized a design pattern to facilitate code reuse across different web pages.
- **OMS** – Developed a Java package to interface to an Order Management System, encapsulating Oracle stored procedure calls through JDBC.
- **CommentLego / ShippingAddressLego** – Designed and developed reusable web components for customer service annotation and shipping address processing (ECML Wallet standard).
- **Merchant Extranet** – Various development roles in a team building a system for managing merchandizing and product information using Java Servlets.
