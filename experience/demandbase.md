# Demandbase — Principal Big Data Engineer

## Context

Designed, developed, and supported Big Data ETL pipelines and infrastructure components, enabling critical business processes supporting account-based marketing.

## Contributions

- **EngagementQube** – Developed Scala Spark pipeline for efficiently generating aggregates of multi-dimensional web engagement events, including reach estimations and time-series analytics.
- **Nutch Crawler** – Leveraged and extended Apache Nutch to efficiently crawl and process millions of URLs on a daily basis. Developed framework to generate crawl metrics.
- **Windmill Document Parser** – Developed an Apache Beam pipeline on GCP Dataflow using Spotify's Scio to extract text from HTML, and perform Named Entity Recognition and keyword extraction with the Stanford NLP library.
- **Intent Pipeline** – Optimized and extended legacy BigQuery pipelines running on Apache Airflow. Drastically reduced BigQuery slot usage and enabled DAGs to meet critical business SLAs.
- **Bumblebee Processing Framework** – Designed and developed GCP Streaming Dataflow using Apache Beam and Spotify Scio for processing synchronization tasks between distributed tenant instances, enabling near real-time updates. Implemented a Pulsar to GCP Pubsub bridge. Developed monitors and alerts integrated into Prometheus with Grafana charts.
