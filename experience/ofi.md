# OFI Corp. — Software Engineer

## Context

OFI ("Office of the Future") was a software project contracting company — not staff augmentation. Engineers worked from OFI's offices on projects delivered as a whole. The major engagement during Jerome's tenure was with General Motors, building software related to the emerging CAFE (Corporate Average Fuel Economy) fuel efficiency standards. This was an era before C++ STL or commercial libraries like RogueWave were generally available — you actually had to implement binary search yourself.

## Contributions

- what: container and iterator classes (binary trees, binary searches) for OFI's base libraries
- stack: C++
- impact: foundational data structures for OFI's base libraries at a time before C++ STL or commercial alternatives like RogueWave were generally available; improved overall application performance

### X/Motif Help System

- stack: C++, X/OSF Motif
- impact: delivered the window-based help system component for the GM/CAFE project application
