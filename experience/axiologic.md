# Axiologic Corporation — Independent Contractor

## Context

Consulted to Wall Street financial firms, designing and developing financial applications and trading systems. Contracted on a 1099 basis through a personal S-corporation. Three sequential client engagements.

## Client: NatWest Markets, Equity Proprietary Trading (August 1996–October 1997)

Developed front-office trading applications to support equity arbitrage trading strategies.

- **PairMonitor** – Designed and developed a real-time analytic tool in C++/X/Motif to monitor hundreds of securities pairs for possible trading signals, featuring real-time sorting, filtering, and historical graphing.
- **Formula / Depends / DataService libraries** – Designed and developed a core C++ computational infrastructure for real-time calculation of arbitrary complex analytic expressions, integrating multiple data sources in a tiered manner. Used across several projects in the group.
- **TimeSeries library** – Designed and developed a C++ template class library implementing container classes for time-related data, supporting multiple time frequencies and holiday schedule handling.
- **Screening Universe** – Developed a periodic Sybase database of all stocks meeting criteria like market cap and traded value, generated from multiple data sources via shell scripts, SQL, FAME procedures, and C++ programs.
- **FameClasses** – Designed and developed a C++ class library for intuitive object-oriented access to FAME time-series databases.
- **PairTrader** – Developed an analytical decision support tool for pairs trading using X/Motif and the Xrt Table Widget, displaying real-time position analytics and Reuters news integration.

## Client: Deutsche-Morgan Grenfell, Equity Arbitrage (October 1995–August 1996)

Developed front-office trading applications, supporting the desk directly.

- **Edgar Expression Scanning** – Designed and developed Perl/Expect scripts to analyze SEC Edgar filings and auto-generate web pages with keyword expressions; notified traders of filings by companies on their watchlists.
- **RPC Proxy Libraries** – Used ONC RPC to create Windows DLL proxy libraries that remotely invoked functions on a different machine.
- **FAME Database Wrapper** – Designed and implemented a C++ class library making FAME's time-series database appear like a relational database.
- **Monte Carlo Pricing Library** – Designed and developed a C++ library for Monte Carlo pricing of derivative securities, extensible to increasingly complex derivative types.
- **Financing Model** – Modified existing applications to incorporate a sophisticated model of stock-loan financing for comparison with back-office calculations.
- **Treasury Note Conversion Factors** – Developed an automated, self-updating database of Treasury Note conversion factors for cheapest-to-deliver calculation.

## Client: Citibank, Global Derivatives (February 1995–October 1995)

Developed software for CDS, a large-scale global derivative trading system.

- **Data Conversion** – Oversaw and led a three-person team to convert data from legacy swap trading systems to CDS for the production release.
- **Database Security System** – Designed a C++ library and Sybase stored procedures to extend the security mechanism of the trading system, providing greater control over data access.
- **ReportWriter** – Supported and extended a GUI report writer application using Teleuse and Sybase SQR.
